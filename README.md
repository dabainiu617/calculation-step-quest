# 计算闯关 App

面向小学三到六年级学生的轻量计算练习 App。第一版采用 React + Vite + localStorage，无需后端和登录，适合部署成一个网页链接分享给学生和家长。

## 功能

- 每天每关 10 题，一题一屏。
- 当前包含第4步到第26步，以及第33步到第40步。
- 所有关卡默认解锁，可按加法、减法、乘法、除法、混合、小数、分数分类任选一关练习。
- 每关显示累计总题数、正确率、练习次数、累计用时和连续达标情况。
- 答对 9 题或以上视为本次达标。
- 连续 2 次练习达标后，标记为该关暂时稳定。
- 每次练习都会统计本轮用时，并计入总用时与平均用时。
- 小数答案直接输入小数；分数答案用 `分子/分母` 输入，例如 `5/6`。
- 结果页集中显示错题订正。
- 进度页适合家长截图发给老师。

## 本地运行

```bash
npm install
npm run dev
```

然后打开终端提示的本地地址。

## 验证核心逻辑

```bash
npm run validate
```

验证内容包括第4-26步、第33-40步题目范围、不重复题、除法合法性、小数/分数判分、9/10 达标、失败后清零、连续两次达标和统计汇总。

## 构建部署

```bash
npm run build
```

生成的 `dist` 目录可以部署到 Vercel、Netlify 或 GitHub Pages。

推荐第一版使用 Vercel 或 Netlify：导入 GitHub 仓库后通常会自动识别 Vite 项目，构建命令为 `npm run build`，输出目录为 `dist`。

## 数据说明

学生昵称、年级、练习记录和关卡进度保存在当前浏览器的 localStorage 中。

这意味着：

- 刷新页面后进度会保留。
- 换设备、换浏览器或清除浏览器数据后，进度不会自动同步。
- 第一版没有老师后台；家长可以打开进度页截图反馈。
